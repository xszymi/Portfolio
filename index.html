Strony Internetowe

80%

HTML (Hypertext Markup Language): To język znaczników używany do strukturyzacji i prezentacji treści na stronach internetowych. HTML definiuje elementy takie jak nagłówki, paragrafy, obrazy i linki, tworząc podstawową strukturę witryny

70%

CSS (Cascading Style Sheets): To język używany do stylizacji stron internetowych. CSS definiuje wygląd i układ elementów HTML, umożliwiając projektowanie atrakcyjnych i spójnych interfejsów użytkownika.

50%

JavaScript: Jest to skryptowy język programowania używany do tworzenia interaktywnych elementów na stronach internetowych. JavaScript działa po stronie klienta, umożliwiając dynamiczne modyfikacje treści i reakcje na interakcje użytkownika.

Języki Programowania

50%

Python: To ogólnego przeznaczenia, wysokopoziomowy język programowania, łatwy w czytaniu i zrozumieniu. Python jest używany do różnorodnych zastosowań, w tym tworzenia aplikacji internetowych, analizy danych, sztucznej inteligencji i automatyzacji zadań.

30%

C#: To język programowania opracowany przez Microsoft. Jest często używany do tworzenia aplikacji na platformę .NET. C# jest obiektowym językiem programowania, oferującym silne typowanie i zaawansowane funkcje.

20%

C++: To język programowania ogólnego przeznaczenia, rozszerzający język C. C++ jest stosowany do tworzenia efektywnych i wydajnych programów, w tym systemów operacyjnych, gier komputerowych i aplikacji z wysoką wydajnością.

Inne umiejętności

30%

Unity to wieloplatformowe środowisko programistyczne do tworzenia gier oraz aplikacji 2D i 3D. Charakteryzuje się łatwością użytkowania, obsługą wielu platform, systemem komponentowym i narzędziami do grafiki oraz fizyki.

60%

GIMP to program do edycji grafiki rastrowej. Pozwala na tworzenie, edycję i obróbkę obrazów, oferując szereg narzędzi i funkcji. GIMP obsługuje wiele formatów plików graficznych, oferuje warstwy, filtry, narzędzia do retuszu oraz możliwość rozszerzania funkcjonalności poprzez wtyczki.

50%

MySQL to otwartoźródłowy system zarządzania relacyjnymi bazami danych, który stosuje model relacyjny. Jest używany do przechowywania i zarządzania danymi w aplikacjach, obsługując język zapytań SQL. MySQL oferuje także mechanizmy transakcji, zapewniając integralność i trwałość operacji na danych.
